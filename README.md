# Flask Todo API

## Features
- RESTful API
- PostgreSQL database
- Docker containerization
- CI/CD with GitHub Actions
- Automated testing

## Quick Start
\`\`\`bash
docker-compose up -d
\`\`\`

## API Endpoints
- `GET /api/health` - Health check
- `GET /api/todos` - Get all todos
- `POST /api/todos` - Create todo
- ...
